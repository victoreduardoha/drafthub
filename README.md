# DraftHub — Competitive Match Draft Platform

Organize partidas personalizadas de Valorant: crie lobbies, faça o draft de jogadores por turno, execute o map ban completo (MD1 / MD3 / MD5) e visualize o resultado final da série.

## Stack

- Next.js 16 App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Zustand (persisted)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
